## 👋 First Steps

### 1. Create Your Account

**What you'll see:**
- Login page with purple AI icon
- Sign up option at bottom

**What to do:**
1. Click "Don't have an account? Sign up"
2. Fill in:
   - Full Name (e.g., "John Doe")
   - Email (e.g., "john@example.com")
   - Password (create a secure one)
3. Click "Sign Up"

**Result:** Instantly logged in and ready to chat!

### 2. Send Your First Message

**The Interface:**
- Sidebar on left (conversations)
- Chat area in center
- Input box at bottom

**Your First Message:**
1. Type in the input box: "Hello! I need help with..."
2. Press `Enter` or click send button (➤)
3. Watch AI respond!

**What happens:**
- Your message appears (blue bubble, right side)
- AI shows "thinking" animation
- AI responds (white bubble, left side)

### 3. Customize Your Experience

**Open Settings:**
- Click "Settings" in sidebar (bottom)

**Choose Your Role:**
- Student - For learning
- Professional - For work
- Developer - For coding
- Researcher - For analysis
- Creative - For artistic work
- General - For everything

**Add Context (Optional but Recommended):**
```
Example: "I'm a web developer working on a React 
e-commerce project. I need help with state 
management and API integration."
```

**Save:** Click "Save Changes"

## 📖 Basic Usage

### Creating Conversations

**When to create new chat:**
- New topic or project
- Different subject
- Clean start needed

**How to create:**
1. Click "New Chat" button (top of sidebar)
2. Fresh conversation starts
3. Old chat is saved

### Switching Conversations

**Desktop:**
- Click any conversation in sidebar
- Loads instantly

**Mobile:**
- Tap menu icon (☰)
- Tap conversation
- Sidebar auto-closes

### Managing Conversations

**Delete a conversation:**
1. Hover over conversation (desktop)
2. Click trash icon (🗑️)
3. Confirm deletion

**Clear current chat:**
1. Click "Clear" button (top right)
2. Confirms before clearing
3. Conversation kept, messages removed

### Getting Better Responses

**Tips:**

✅ **Be Specific:**
```
Good: "How do I create a responsive navbar in React 
with mobile menu?"

Not ideal: "Help with React"
```

✅ **Use Settings:**
- Set your role
- Add context
- Get tailored responses

✅ **Ask Follow-ups:**
- Continue conversation
- Reference previous messages
- Build on responses

## ❓ Quick FAQ

### Can I use it on my phone?
✅ Yes! Fully responsive and touch-optimized.

### Do I need internet?
⚠️ UI works offline, but AI responses need internet (when connected to backend).

### Where is my data stored?
📱 Locally in your browser (localStorage). Private and secure.

### Can I export conversations?
⏳ Not yet, but it's on the roadmap!

### Is it free?
✅ The frontend is free and open source. AI API costs depend on which service you integrate.
